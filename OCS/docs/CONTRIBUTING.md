# 🤝 贡献指南

感谢你对 Open-code-Studio 开源项目的关注！我们欢迎任何形式的贡献。

---

## 💬 如何参与

### 报告问题

如果你发现了 Bug 或有功能建议：

1. 访问对应项目的 GitHub Issues 页面
2. 搜索是否已有相似的 issue
3. 如果没有，创建一个新的 issue，包含：
   - 清晰的标题和描述
   - 复现步骤（如果是 Bug）
   - 系统环境和版本信息

### 贡献代码

1. **Fork** 对应项目的仓库
2. **Clone** 你的 fork 到本地
3. 创建一个新的分支：`git checkout -b feature/your-feature`
4. 编写代码，遵循项目的代码风格
5. 提交前运行测试
6. 提交你的更改并推送到你的 fork
7. 在 GitHub 上创建 **Pull Request**

### 改进文档

文档改进是最受欢迎的贡献之一！如果你发现：

- 拼写或语法错误
- 缺失的信息
- 可以更清晰的表述

欢迎直接提交 PR 来改进。

---

## 📐 代码规范

- 遵循项目已有的代码风格
- 为新功能编写适当的文档
- 提交信息使用清晰的描述
- PR 尽量保持小而聚焦

---

## 📮 沟通渠道

- **GitHub Discussions**：[讨论区](https://github.com/orgs/Open-code-Studio/discussions)
- **GitHub Issues**：通过各项目的 Issue 报告问题

我们期待你的参与！🎉
