# 👋 加入我们

Open-code-Studio 始终欢迎志同道合的开发者加入！

---

## 🚪 如何加入

1. **关注我们的项目**：在 GitHub 上 Star 你感兴趣的项目
2. **参与讨论**：加入 [GitHub Discussions](https://github.com/orgs/Open-code-Studio/discussions)
3. **提交贡献**：参考 [贡献指南](#/contributing) 开始贡献
4. **联系我们**：通过 [联系方式](#/contact) 与我们沟通

---

## 🌱 从哪开始

如果你是第一次参与，建议从以下入手：

1. 改进或翻译文档
2. 帮忙回复 Discussions 中的问题
3. 测试最新版本并提供反馈

> 无论你是经验丰富的开发者，还是刚入门的新手，这里都有适合你的位置！
