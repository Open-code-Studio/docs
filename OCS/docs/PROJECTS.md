## 🎮 JMCL — Java Minecraft Launcher

一款开源、跨平台的 **Minecraft: Java Edition** 启动器。

- **技术栈**：Kotlin, Compose Multiplatform
- **平台支持**：Windows / macOS / Linux
- **许可证**：开源
- **GitHub**：[Open-code-Studio/JMCL](https://github.com/Open-code-Studio/JMCL)
- **文档**：[JMCL 文档站](https://open-code-studio.github.io/docs/JMCL/page/index.html#/)

### 主要特性

- 🚀 快速启动，专为 Minecraft 优化
- 🖥️ 原生跨平台体验（Windows / macOS / Linux）
- 🔌 可扩展的插件架构
- 🌍 多语言本地化支持
