# ℹ️ 关于

**Open Web Blog Kit** 是一个轻量级 Material Design 3 博客框架。

---

## 设计理念

- **纯静态** — 不需要 Node.js 构建步骤
- **JSON 驱动** — 配置放在 `config.json`，无需改 JS
- **MD3 主题** — 完整的 Material Design 3 调色板 + 动态主题色
- **模块化** — JS 模块职责分明（app/page/extensions/theme/transitions）

---

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3（8 文件） | 页面结构与 MD3 样式 |
| Vanilla JS（~18KB 核心） | app.js + page.js + extensions.js + theme.js |
| [marked.js](https://marked.js.org/) | Markdown 渲染 |
| [highlight.js](https://highlightjs.org/) | 代码高亮 |
