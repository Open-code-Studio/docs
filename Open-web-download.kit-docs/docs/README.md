# 📦 Open-web-download.kit

轻量级 Material Design 3 软件下载站框架。纯静态，JSON 驱动，零依赖构建。

---

## 核心特性

- 🎨 **MD3 动态主题** — 从单一 hex 色值自动计算完整颜色体系
- 📦 **多项目支持** — 同时展示多个软件项目
- 🏷️ **分类筛选** — 按类别过滤 + 搜索框
- ⬇️ **详情面板** — 侧栏式详情页，含下载文件列表和更新日志
- 🖥️ **平台检测** — 自动识别用户系统，推荐对应版本
- 🌓 **主题切换** — 亮/暗模式，带平滑过渡
- 📂 **双协议兼容** — 同时支持 HTTP 服务器和 `file://` 本地打开
