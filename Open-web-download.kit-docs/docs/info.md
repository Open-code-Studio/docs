# ℹ️ 关于

**Open-web-download.kit** 是一个轻量级 Material Design 3 软件下载站框架。

---

## 设计理念

- **JSON 驱动** — 配置集中在 config.json + projects.js
- **双协议兼容** — file:// 和 HTTP 均可正常工作
- **动态颜色** — 只写一个 hex，全站颜色自动生成
- **纯静态** — 不需要 Node.js 构建

---

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构 + MD3 设计系统（25KB） |
| Vanilla JS | 核心逻辑 20KB（main + config + projects） |
| Google Material Symbols | 图标字体 |
| Google Fonts (Roboto + Noto Sans SC) | 字体 |
